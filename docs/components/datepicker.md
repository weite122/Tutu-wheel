# DatePicker 日历组件

当你需要显示日历可以用该组件，组件实现表单和日历的双向绑定，可以选择任意年月日，当天日期是灰色背景，选中日期有绿色边框。

<datepicker-demo-1 />
</ClientOnly>

## Attributes

|      参数      |      说明      |   类型   | 可选值 | 默认值 |
| :------------: | :------------: | :------: | :----: | :----: |
|     firstDayOfWeek     |     每星期的第一天     |  Number   |   —    |   1    |
|    value  |  日期的初始值  |  Date   |   —    |   New Date()    |
| scope | 显示日期的范围 |  Array  |   —    | -  |
